D Capability strip
What the platform gives you, at a glance

Dense and scannable. Works as a strip beneath A or B.

Kyber

A Kubernetes Agent Management Platform

⚙

Agents as k8s resources

Each agent is a first-class custom resource, reconciled like any workload.

🧠

Persistent memory

A sidecar saves each agent's memory so it survives restarts and moves.

🔗

Signed webhook dispatch

Work is routed to agents as HMAC-signed webhook envelopes.

♻

Self-healing pods

Crashed or OOM'd agents come back on their own, state intact.

🧩

Harness-agnostic

Claude Code today; any agent runtime the same way.

🌍

Multi-cluster fleet

One platform across clusters — home lab to cloud.
